# Contributing to VaseFX

Thanks for contributing.

## Development Setup

1. Install Node.js 18+.
2. Install dependencies:

```powershell
npm install
```

3. Run local server:

```powershell
npm run dev
```

4. Open http://localhost:8080

## Build

```powershell
npm run build
```

Build output is written to `build/index.html`.

## Pull Request Guidelines

- Keep changes focused and small.
- Describe behavior changes and include screenshots for UI changes.
- Avoid unrelated refactors in the same PR.
- Preserve existing coding style.
- Confirm build passes before opening a PR.

## Commit Message Guidance

Use clear, imperative commit messages, for example:

- `fix: preserve sculpt state after randomize`
- `docs: add release notes section`

## Reporting Bugs

Use the bug report template and include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Browser and OS versions
